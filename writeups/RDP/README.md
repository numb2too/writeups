# RDP
Remote Desktop Protocol（遠端桌面通訊協定）

## 說明
RDP 全名是 **Remote Desktop Protocol（遠端桌面通訊協定）**，簡單說就是用來 **從一台電腦遠端操作另一台 Windows 電腦** 的協定。

重點整理：

1. **用途**

   * 讓使用者可以在家或辦公室的電腦上操作另一台電腦，就像坐在那台電腦前一樣。
   * 可以遠端開啟應用程式、管理檔案、甚至做維護工作。

2. **常用環境**

   * Windows 系統內建支援 RDP（「遠端桌面連線」程式）。
   * Linux / Mac 也可以透過 RDP 客戶端軟體連到 Windows 主機。

3. **通訊方式**

   * RDP 使用 **TCP 3389 埠**（預設），並加密傳輸資料。
   * 可設定帳號密碼驗證，必要時也可以搭配多因素認證。

4. **安全注意事項**

   * 因為 RDP 可以完全操作目標電腦，所以若帳號密碼被破解，攻擊者就能完整控制電腦。
   * 實務上會建議：

     * 不要直接暴露 RDP 到公開網路
     * 使用強密碼
     * 搭配 VPN 或防火牆限制來源 IP

所以你剛剛的 Hydra 命令，其實就是對目標的 **RDP 登入介面做密碼猜測攻擊**。
